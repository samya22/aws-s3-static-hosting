# AWS S3 Static Website Hosting

## Project Overview
This project demonstrates static website hosting using Amazon S3 and Python boto3 SDK.

## Technologies Used
- Amazon S3
- Python
- boto3
- HTML
- CSS

## Features
- Static website hosting
- Automated file upload using boto3
- Simple web interface

## Project Files
- index.html
- upload.py
- requirements.txt
- architecture.txt

## Architecture
User Browser → Amazon S3 Bucket → Static Website Hosting

## Outcome
Successfully hosted a static website using AWS S3.
